# 🌍 Global Temperature Dashboard (1850-2010)  

## 📊 Impact of These Visualizations  

Understanding climate trends is crucial in tackling climate change. This dashboard provides **insights into global temperature variations** from 1850 to 2010 across key countries—Brazil, Philippines, China, Sweden, India, the U.S., and Italy.  

🔹 **Why It Matters?**  
- Identifies long-term warming trends across regions 🌡️  
- Highlights temperature fluctuations using **filled maps, line charts, treemaps, and clustered column charts** 📈  
- Interactive dashboard with **live data insights** for better exploration 🖥️  

## 📌 **Visualizations Included**  

✅ **Filled Map (2010)** – Displays **average temperature** by country 🌍  
✅ **Line Chart** – Shows **median temperature trends** over time 📉  
✅ **Treemap (2010)** – Illustrates **temperature distribution by country & month** 🌳  
✅ **Clustered Column Chart (2010)** – Compares **temperature variations by month across countries** 📊  
✅ **Custom Visualization** – Uncovering unique climate insights with a new perspective 🔎
